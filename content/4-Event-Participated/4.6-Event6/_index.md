## BUILDING AGENTIC AI - Context Optimization with Amazon Bedrock

### Event Information

&emsp; **Event Name:** Building Agentic AI - Context Optimization with Amazon Bedrock <br>
&emsp; **Date:** Friday, December 5, 2025 <br>
&emsp; **Location:** 26th Floor, Bitexco Financial Tower <br>
&emsp; **Main Topic:** Build autonomous AI agents with Amazon Bedrock through hands-on techniques and real-world use cases <br>

### Event Overview

This hands-on workshop focused on building autonomous AI agents using Amazon Bedrock. The event brought together experts from AWS, Azure, and the Kubernetes community to share insights on agentic AI development, context optimization techniques, and real-world implementation strategies. Attendees had the opportunity to learn about AWS Bedrock Agent Core, explore practical use cases, and participate in a hands-on workshop building agentic workflows.

### Key Takeaways

**Technical Insights:**

1. **Agent Core Architecture**: Understanding the fundamental components of Amazon Bedrock Agents including runtime, memory, and tool integration

2. **Context Optimization**: Techniques for optimizing context windows to improve agent performance and reduce costs

3. **Agentic Orchestration**: Patterns for building multi-step autonomous workflows with proper state management

4. **Real-World Applications**: Practical use cases demonstrating how enterprises are leveraging agentic AI

**Practical Skills:**
- Hands-on experience building agents with Amazon Bedrock
- Understanding of context optimization strategies
- Knowledge of production deployment best practices

### Personal Reflections

This workshop provided valuable insights into the rapidly evolving field of agentic AI. The combination of theoretical presentations from AWS experts and hands-on practice with CloudThinker's tools gave a comprehensive understanding of how to build autonomous AI agents. The venue at Bitexco Financial Tower offered excellent networking opportunities with professionals from various backgrounds including AWS, Azure, and Kubernetes communities.

The L300-level session on context optimization was particularly valuable, providing advanced techniques that can be immediately applied to improve agent performance in production environments. The hands-on workshop allowed attendees to apply these concepts in real-time, reinforcing the learning experience.
